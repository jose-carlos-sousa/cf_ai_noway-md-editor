# PROMPTS

## Example prompts

- UI improvement / polish:
  - "Review the following markdown UI text and suggest improvements to layout, labels, and microcopy to make it clearer and more accessible. Provide a short changelog and code snippets for minimal tailwind adjustments."

- Accessibility audit:
  - "Scan this component's HTML/JSX and recommend accessibility improvements (aria attributes, color contrast, keyboard navigation). Return concise code patches."
  
- Security review prompt:
  - "Quickly scan this code snippet for obvious secrets, unsafe environment variable usage, or client-side exposure of keys; recommend fixes and safe storage patterns."

- Architecture guidance:
  - "Given this use case, recommend which Cloudflare services (e.g., Workers KV, Durable Objects, R2) are best suited for storing and retrieving key-value data. Explain trade-offs and provide example usage patterns."
